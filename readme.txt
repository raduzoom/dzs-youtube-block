A professional Gutenberg block for embedding YouTube videos with advanced customization, responsive design, and seamless editor integration.

== Description ==

DZS YouTube Block is a comprehensive WordPress plugin that enhances content creation with a professional YouTube block for the Gutenberg editor. It provides a robust and customizable solution for embedding YouTube videos with advanced features and performance-focused implementation.

The plugin is designed for developers, content creators, agencies, and businesses who want more control and better presentation than standard YouTube embeds.

= Key Features =

* Native Gutenberg block integration
* Custom block inspector controls
* Multiple player themes and styles
* Responsive design for all devices
* Custom CSS support
* Optimized and lightweight embedding
* Clean and extensible codebase
* Elementor widget integration

= Perfect For =

* Content creators presenting video content
* Developers building custom WordPress themes
* Agencies creating client websites
* Bloggers enhancing posts with videos
* Businesses showcasing product demos

= Why Choose DZS YouTube Block? =

Unlike basic YouTube embeds, this plugin offers a professional appearance, improved user experience, and extensive customization options while remaining future-proof and compatible with modern WordPress versions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dzs-video-block-for-youtube` directory, or install the plugin directly from the WordPress Plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Add the DZS YouTube Block from the Gutenberg block inserter.

== Frequently Asked Questions ==

= Is this plugin compatible with my theme? =
Yes. The plugin works with all WordPress themes that support the Gutenberg editor.

= Does this plugin work with page builders? =
Yes. It is compatible with most page builders, including Elementor via a dedicated widget.

= Is the plugin responsive? =
Yes. All embedded videos are fully responsive and adapt to all screen sizes.

= Can I use this with YouTube playlists? =
Yes. The plugin supports both single videos and playlists.

= Is there a limit on how many videos I can embed? =
No. You can embed unlimited videos across posts and pages.
